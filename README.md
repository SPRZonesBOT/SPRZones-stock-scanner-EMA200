# SPRZones - EMA200 Stock Scanner

Indian market stock scanner for **200 EMA Breakout** on **1H, 4H, Daily** timeframes.

## 🚀 Features
- Scans Top 100 Nifty stocks.
- Detects EMA Crossover + Bullish Patterns (Engulfing, Hammer, Dragonfly).
- Checks Fundamentals (PE, ROE, Debt, Margin, Growth).
- Streamlit based - Accessible on any device (Mobile/Laptop/Tablet).

## 🛠️ How to Run Locally
```bash
pip install -r requirements.txt
streamlit run app.py
